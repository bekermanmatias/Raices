¿Qué es una raíz de función?

Conceptos fundamentales para análisis numérico

Definición de Raíz

Una raíz (o cero) de una función f(x) es un valor de x tal que:

f(x) = 0

Ejemplo:

f(x) = x² - 5x + 6

Las raíces son: x = 2 y x = 3

Verificación: f(2) = 0 y f(3) = 0

Interpretación Geométrica

Las raíces de una función corresponden a los puntos de intersección de la gráfica de f(x) con el eje X.

Características:

  • • Puntos donde y = 0
  • • Pueden ser reales o complejas
  • • Una función puede tener múltiples raíces
  • • Algunas funciones no tienen raíces reales

Método Manual: Fórmula Cuadrática

Encontrando las raíces de f(x) = x² - 5x + 6 = 0 paso a paso

1

Identificar coeficientes

a = 1, b = -5, c = 6

2

Aplicar fórmula cuadrática

x = (-b ± √(b² - 4ac)) / 2a

3

Calcular

x = (5 ± √(25 - 24)) / 2

x = (5 ± 1) / 2

4

Resultado final

x = 2 y x = 3

Visualización

Código Python para Cálculo de Raíces

Método 1: Usando NumPy

import numpy as np

# Para polinomios: x² - 5x + 6 = 0
coeficientes = [1, -5, 6]  # [a, b, c] para ax² + bx + c
raices = np.roots(coeficientes)
print("Raíces:", raices)

# Resultado: [3. 2.]

Método 2: Usando SciPy (Más preciso)

from scipy.optimize import fsolve
import numpy as np

def ecuacion(x):
    return x**2 - 5*x + 6

# Encontrar raíces cerca de valores iniciales
raiz1 = fsolve(ecuacion, 1)  # Busca cerca de x=1
raiz2 = fsolve(ecuacion, 4)  # Busca cerca de x=4

print(f"Raíz 1: {raiz1[0]:.6f}")
print(f"Raíz 2: {raiz2[0]:.6f}")

Método 3: Usando SymPy (Cálculo simbólico)

from sympy import symbols, solve, diff
import sympy as sp

x = symbols('x')
ecuacion = x**2 - 5*x + 6

# Resolver simbólicamente
raices = solve(ecuacion, x)
print("Raíces exactas:", raices)

# Evaluar numéricamente
for raiz in raices:
    print(f"x = {float(raiz):.6f}")

Método 4: Newton-Raphson (Implementación manual)

def newton_raphson(f, df, x0, tolerancia=1e-6, max_iter=100):
    """
    Método de Newton-Raphson para encontrar raíces
    f: función
    df: derivada de la función
    x0: valor inicial
    """
    x = x0
    for i in range(max_iter):
        fx = f(x)
        if abs(fx) < tolerancia:
            return x, i+1
        dfx = df(x)
        if dfx == 0:
            return None, i+1
        x = x - fx/dfx
    return x, max_iter

# Definir función y su derivada
def f(x):
    return x**2 - 5*x + 6

def df(x):
    return 2*x - 5

# Encontrar raíces
raiz1, iter1 = newton_raphson(f, df, 1)
raiz2, iter2 = newton_raphson(f, df, 4)

print(f"Raíz 1: {raiz1:.6f} (en {iter1} iteraciones)")
print(f"Raíz 2: {raiz2:.6f} (en {iter2} iteraciones)")
