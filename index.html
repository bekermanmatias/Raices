Raíces de Funciones

Concepto y cálculo paso a paso

Definición de Raíces de Funciones

Dada una función f(x) continua en un entorno, decimos que aquellos valores xi tales que f(xi) = 0, son las raíces de la función.

$$f(x_i) = 0$$

¿Qué significa "función continua"?

Definición de continuidad: Una función se dice continua en un intervalo "I" dado si es continua en todo punto "a" del interior de este intervalo.

Se dice que una función f es continua en el número "a" si y sólo si se cumplen las tres condiciones siguientes:

i

$$ f(a)\text{ existe}$$

La función está definida en el punto a

ii

$$\lim_{x \to a} f(x) \text{ existe}$$

El límite existe cuando x se acerca a a

iii

$$\lim_{x \to a} f(x) = f(a)$$

El límite es igual al valor de la función

¿Por qué es importante la continuidad?

Las funciones continuas garantizan que no hay "saltos" o "huecos" en la gráfica. Esto es fundamental para encontrar raíces, ya que si una función continua cambia de signo en un intervalo, debe cruzar el eje X al menos una vez.

Existen principalmente dos tipos de raíces según el tipo de función:

A

Polinomios

Funciones algebraicas donde la variable aparece elevada a potencias enteras no negativas.

Ejemplo:

$$ax^2 + bx + c = 0$$

Ecuación cuadrática

Características:

  • Tienen un número finito de raíces
  • Se pueden resolver analíticamente
  • Métodos: fórmula cuadrática, factorización
B

Ecuaciones Implícitas

Funciones que combinan términos algebraicos y trascendentes, donde la variable no se puede despejar fácilmente.

Ejemplo:

$$\tan(x) - x = 0$$

Ecuación trascendente

Características:

  • Pueden tener infinitas raíces
  • Requieren métodos numéricos
  • Métodos: Newton-Raphson, bisección

Ejemplos

A

Raíces de Polinomios

Ecuación cuadrática:

$$2x^2 - 8x + 6 = 0$$

Paso 1: Identificar coeficientes: a=2, b=-8, c=6

Paso 2: Aplicar fórmula cuadrática:

$$x = \frac{8 \pm \sqrt{64-48}}{4} = \frac{8 \pm 4}{4}$$

Resultado: x₁ = 3, x₂ = 1

Verificación:

f(3) = 2(3)² - 8(3) + 6 = 0 ✓

f(1) = 2(1)² - 8(1) + 6 = 0 ✓

Visualización

La parábola intersecta el eje X en x = 1 y x = 3

B

Ecuaciones Implícitas

Ecuación trascendente:

$$\tan(x) - x = 0$$

Problema: No se puede resolver analíticamente

Método: Métodos numéricos (Newton-Raphson)

Algunas raíces aproximadas:

  • x₁ ≈ 0
  • x₂ ≈ ±4.493
  • x₃ ≈ ±7.725
  • x₄ ≈ ±10.904

Características:

• Infinitas raíces simétricas

• Se intersectan y = tan(x) y y = x

Visualización

Código Python para Cálculo de Raíces

Método 1: Usando NumPy

import numpy as np

# Para polinomios: x² - 5x + 6 = 0
coeficientes = [1, -5, 6]  # [a, b, c] para ax² + bx + c
raices = np.roots(coeficientes)
print("Raíces:", raices)

# Resultado: [3. 2.]

Método 2: Usando SciPy (Más preciso)

from scipy.optimize import fsolve
import numpy as np

def ecuacion(x):
    return x**2 - 5*x + 6

# Encontrar raíces cerca de valores iniciales
raiz1 = fsolve(ecuacion, 1)  # Busca cerca de x=1
raiz2 = fsolve(ecuacion, 4)  # Busca cerca de x=4

print(f"Raíz 1: {raiz1[0]:.6f}")
print(f"Raíz 2: {raiz2[0]:.6f}")

Método 3: Usando SymPy (Cálculo simbólico)

from sympy import symbols, solve, diff
import sympy as sp

x = symbols('x')
ecuacion = x**2 - 5*x + 6

# Resolver simbólicamente
raices = solve(ecuacion, x)
print("Raíces exactas:", raices)

# Evaluar numéricamente
for raiz in raices:
    print(f"x = {float(raiz):.6f}")

Método 4: Newton-Raphson (Implementación manual)

def newton_raphson(f, df, x0, tolerancia=1e-6, max_iter=100):
    """
    Método de Newton-Raphson para encontrar raíces
    f: función
    df: derivada de la función
    x0: valor inicial
    """
    x = x0
    for i in range(max_iter):
        fx = f(x)
        if abs(fx) < tolerancia:
            return x, i+1
        dfx = df(x)
        if dfx == 0:
            return None, i+1
        x = x - fx/dfx
    return x, max_iter

# Definir función y su derivada
def f(x):
    return x**2 - 5*x + 6

def df(x):
    return 2*x - 5

# Encontrar raíces
raiz1, iter1 = newton_raphson(f, df, 1)
raiz2, iter2 = newton_raphson(f, df, 4)

print(f"Raíz 1: {raiz1:.6f} (en {iter1} iteraciones)")
print(f"Raíz 2: {raiz2:.6f} (en {iter2} iteraciones)")
